# Liquent Reports

This directory is reserved for local backtest reports generated by Liquent CLI
tools.

Generated reports are local artifacts and must not be committed.

Tracked:

- `reports/README.md`

Ignored:

- generated Markdown reports
- generated JSON reports
- any local analysis output

## Safety

Reports are generated from local files only.

The current CLI does not:

- connect to exchanges,
- perform live trading,
- perform paper trading,
- download data,
- use API keys,
- schedule automated runs.

Reports are descriptive research/backtesting artifacts only and are not
investment advice.

## Typical usage

```bash
python -m liquent.cli.backtest_mid_breakout \
  --csv tests/fixtures/strategy_mid_breakout_long.csv \
  --output reports/mid_breakout_example.md \
  --symbol TESTUSDT \
  --exchange synthetic \
  --asset-class crypto \
  --overwrite
```

Do not commit generated files from this directory.
